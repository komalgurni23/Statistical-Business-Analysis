# 📊 Statistical Business Analysis

## Project Overview

This project performs statistical analysis on business sales data to discover meaningful insights. The analysis includes descriptive statistics, hypothesis testing, confidence intervals, correlation analysis, and linear regression.

---

## Objectives

- Calculate descriptive statistics
- Analyze data distribution
- Perform hypothesis testing
- Calculate confidence intervals
- Analyze correlation between Marketing Spend and Sales
- Build a linear regression model
- Generate business insights

---

## Dataset

File:

business_data.csv

Columns

- Date
- Product
- Quantity
- Price
- Customer_ID
- Region
- Marketing_Spend
- Total_Sales
